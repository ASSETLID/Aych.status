Dose3 / Distcheck

Dose is a library and a collection of tools to perform la large spectrum of analysis on package repositories.

Distcheck is a tool to determine whether packages of the repository can be installed.

  • Homepage : dose.gforge.inria.fr
  • Authors : Pietro Abate
  • Code : git clone https://gforge.inria.fr/git/dose/dose.git
  • Documentation : quickstart
